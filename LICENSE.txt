CRUD SQL Server Management System
نظام إدارة قواعد البيانات لـ SQL Server

This software is designed to provide a comprehensive database management
system for SQL Server with CRUD operations, built with Electron and 
Quasar Framework.

هذا البرنامج مصمم لتوفير نظام شامل لإدارة قواعد البيانات لـ SQL Server
مع عمليات CRUD، مبني باستخدام Electron و Quasar Framework.
